# Robotic-Arm
A documentation of my robotic arm project.

Purpose:
- Learn robotics engineering
- Improve CAD skills
- Understand electronic systems
- Understand mechanical properties (torque, backlash, stiffness)

Goal: Create a teleoperated robotic arm with a PS4 controller

## Description of Arm (V1)
- 5 DOF (base rotation, shoulder pitch, elbow pitch, wrist pitch, gripper open/close)
- Reach: ~ 350 m
- Payload: 100-200 g (phone-sized object as a benchmark)
- Budget: Minimum of $100, target of $250, stretch of $400
- Material: PETG filament
- Controller: PS4 controller
- Processor: ESP32
- Communication: Bluetooth

## Success Criteria
- Arm can be controlled by a PS4 controller
- Arm can reliably move through all joints
- Arm can smoothly grasp lightweight objects

## Future Expansions
- Higher torque servos
- Transmission Systems (e.g. harmonic drives, planetary gearboxes, cycloidal drive)
- Inverse kinematics
- Additional rotational axis
- Computer vision
